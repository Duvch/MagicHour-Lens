# MagicHour Lens

Transform any image on the web — face swap, style transfer, AI edits & more. A Chrome extension powered by [MagicHour AI](https://magichour.ai?ref=falconhacks).

## Features

- **Style Transfer** — Transform images into anime, oil painting, pixel art, etc.
- **Face Swap** — Swap faces in any image on the web
- **Auto Face Swap** — Automatically face-swap images on every page you visit
- **AI Edit** — Describe edits in natural language
- **Remove Background** — One-click background removal
- **Upscale** — AI-powered image upscaling
- **Change Clothes** — Virtual try-on with reference images

## Getting Started

1. Install the extension in Chrome (`chrome://extensions` > Load unpacked)
2. Click the extension icon > Settings
3. Add your [MagicHour API key](https://magichour.ai/developer?ref=falconhacks)
4. Click any image on a page and hit Transform
